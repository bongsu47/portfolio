# 3. 업무용 보고서 자동 생성하기

### 황매니저의 고민

황매니저는 요즘 LLM 덕분에 보고서를 쓰는 속도가 눈에 띄게 빨라졌습니다.\
빈 화면을 바라보는 시간도, 첫 문장을 고민하는 시간도 줄었습니다.

버튼을 누르면 생각보다 훨씬 그럴듯하게 초안이 완성됩니다.

그런데 이상합니다.\
보고서는 이미 완성된 것 같은데, 일은 끝나지 않습니다.

파일을 다시 열어 회사 로고에 맞게 정리하고,\
정해진 템플릿 구조에 맞춰 항목을 옮기고,\
결국 ‘우리 회사 문서’로 다시 태어나게 만듭니다.

빠르게 시작했지만, 마지막은 여전히 수작업입니다.

황매니저는 문득 생각합니다.

_“처음부터 우리 회사 형식으로 만들어지면 안 될까?”_

이번 장에서는 LLM이 만든 단순한 마크다운 결과를 넘어,\
**우리 회사 전용 템플릿에 맞춘 보고서로 생성하는 방법**을 살펴봅니다.

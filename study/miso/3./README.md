# 3. 우리 회사 캐릭터로 이미지 생성하기

### 홍매니저의 고민

홍매니저는 매번 홍보 이미지를 만들 때마다 같은 작업을 반복합니다.\
샘플 이미지를 Gemini에 첨부하고, 원하는 느낌을 다시 설명하고, 캐릭터 설정을 또 적어 넣습니다.

<figure><img src="../../../.gitbook/assets/image 418 (1).png" alt="" width="375"><figcaption></figcaption></figure>

_“왜 매번 똑같은 이미지를 첨부해야 하지?”_

이미지는 금방 만들어지지만, 준비 과정이 번거롭습니다.\
사실 홍매니저가 원하는 건 단 하나입니다.

샘플을 매번 올리지 않아도,\
“이런 느낌으로 만들어줘”라고 요청하면\
자동으로 우리 회사 캐릭터를 기반으로 이미지가 생성되는 것.

이번 장에서는 우리 회사 캐릭터를 기준으로,\
요청만으로 원하는 이미지를 자동 생성하는 워크플로우를 구성해봅니다.

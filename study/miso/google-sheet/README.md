# 2. 설문 데이터로 VOC 분석하기

### 이매니저의 고민

이매니저는 프로 축구 구단의 운영 매니저로, 홈경기 운영과 팬 경험 관리를 담당하고 있습니다. 그의 중요한 업무 중 하나는 경기 관람 만족도 설문을 분석해 다음 경기 운영을 개선하는 것입니다.

설문 응답은 Google Sheet에 자동으로 쌓입니다. 추천 점수나 재방문 의사 같은 객관식 항목은 비교적 쉽게 정리할 수 있지만, 문제는 주관식 의견입니다.

팬들은 경기장 시설, 먹거리, 티켓 예매, 이벤트, 관람 안내 등 다양한 의견을 남깁니다. 이매니저는 수백 개의 의견을 하나씩 읽으며 비슷한 내용을 묶어 정리하지만, 응답이 늘어날수록 같은 의미의 의견이 다르게 표현되거나 여러 주제가 섞여 있어 분류가 점점 어려워집니다.

_“이 많은 의견들을 사람이 직접 분류해야 할까?”_

팬들의 목소리는 중요하기 때문에 담당자인 이매니저가 의견을 직접 확인하는 과정은 여전히 필요합니다. 다만 사람이 모든 의견을 하나씩 읽고 비슷한 내용끼리 분류하는 작업까지 전부 수작업으로 진행하는 방식에는 분명한 한계가 있습니다.

그래서 이매니저는 생각합니다.

“Google Sheet에 있는 설문 데이터를 활용해 AI가 팬들의 주요 VOC를 자동으로 정리해 줄 수는 없을까?”

이번 장에서는 Google Sheet에 쌓인 설문 데이터를 활용해 팬들의 주관식 의견을 자동으로 분류하고 주요 VOC를 정리하는 방법을 살펴봅니다.

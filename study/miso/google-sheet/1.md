## 시작 노드

시작 노드는 사용자의 요청이나 데이터를 처음으로 입력받는 노드입니다.

엑셀 데이터를 사용하는 경우에는 파일 변수를 통해 사용자가 파일을 첨부할 수 있지만, 레벨 1에서는 Google Sheet 도구를 통해 실제 시트의 데이터를 직접 가져오기 때문에 사용자가 별도로 첨부해야 할 데이터는 없습니다.

## Google Sheet 도구 노드

Google Sheets는 표 형태로 데이터를 정리하고 관리할 수 있는 구글의 온라인 스프레드시트 도구입니다. 여러 사람이 동시에 데이터를 입력하고 확인할 수 있어 업무 관리, 보고서 작성, 고객 데이터 정리 등 다양한 업무에 널리 활용됩니다.

#### 도구 기능 알아보기

구글 시트 도구는 총 **5가지 기능**을 제공합니다. 레벨 1에서는 `values.get` 을 다뤄보겠습니다.

### STEP 2. 데이터 주소 확인하기

Google Sheet에서 데이터를 가져오려면 배달을 시킨다고 생각해보면 이해하기 쉽습니다.&#x20;

우리가 음식을 주문할 때 **우리 집 주소를 정확하게 입력하지 않으면 배달이 제대로 올 수 없겠죠**.&#x20;

Google Sheet도 마찬가지입니다. 원하는 데이터를 가져오기 위해서는 먼저 아래의 3가지 정보를 먼저 확인해야 합니다.

<figure><img src="../../../.gitbook/assets/image (507).png" alt="" width="563"><figcaption></figcaption></figure>

{% hint style="info" %}
#### 데이터를 가져오기위해 알아야할 필수 정보

* **Google Sheet ID**
  * Google Sheet마다 고유한 ID가 있습니다. 시트 주소(URL)에서 확인할 수 있습니다.
  * Google Sheet 주소에서 `/d/`와 `/edit` 사이에 있는 값이 **Sheet ID**입니다.
    *   예를 들어, 다음 주소가 있을 때 Sheet ID는 `1_Bc7DcA4VD4uoN29w8AU2x4rRWJbZ8vZSMRPYFGN0kQ`입니다.

        <pre data-full-width="true"><code>https://docs.google.com/spreadsheets/d/1_Bc7DcA4VD4uoN29w8AU2x4rRWJbZ8vZSMRPYFGN0kQ/edit?gid=0#gid=0 
        </code></pre>
* **시트 이름 (Sheet Name)**
  * 하나의 Google Sheet 파일 안에는 여러 개의 시트 탭이 있을 수 있습니다. 이때 데이터를 가져오려는 **탭의 이름**을 정확히 알아야 합니다.
  * 시트 이름은 Google Sheet 하단에 보이는 탭 이름을 그대로 입력하면 됩니다. 띄어쓰기나 특수문자, 대소문자가 동일하게 입력해야 합니다.
* **데이터 범위 (Range)**
  * 가져올 데이터가 시트 안의 **어느 위치에 있는지** 범위를 지정해야 합니다. 데이터 범위는 보통 열과 행을 조합해서 표시합니다.
  * 만약 `설문응답` 시트에서 A열부터 D열까지 가져오고 싶다면 범위는 `A:D`처럼 지정할 수 있습니다.\
    특정 구간만 가져오고 싶다면 `A2:D20`처럼 시작 셀과 끝 셀을 함께 지정하면 됩니다.
{% endhint %}

예를 들어, 예제 시트를 기준으로 보면 `data`라는 시트에 A열부터 AH열까지 설문 정보가 들어 있다고 가정할 수 있습니다. 이 경우 필요한 정보는 다음과 같이 정리할 수 있습니다.

* **시트 이름(Sheet Name)**: `data`
* **데이터 범위(Range)**: `A:AH`

즉, `data` 시트의 A열부터 AH열까지 있는 모든 설문 응답 데이터를 가져오려면 범위를 `data!A:AH`로 지정하면 됩니다.

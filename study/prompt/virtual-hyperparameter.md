# 가상 하이퍼파라미터

하이퍼파라미터(Hyperparameter)는 소프트웨어가 제대로 작동하도록 사용자가 미리 정해주는 설정 값을 말합니다.

쉽게 말해, 소프트웨어가 어떻게 동작할지를 미리 조절할 수 있는 조작 버튼이나 설정 항목 이라고 생각하시면 됩니다.

예를 들어 슈퍼마리오와 같은 게임에서 중력에 대한 하이퍼파라미터 값을 아주 낮게 설정한다면 마리오가 훨씬 높게 그리고 오래 점프할 수 있게 됩니다. 반대로 중력을 높게 설정한다면 점프를 낮게 그리고 짧게 하게 됩니다.

GenAI와 대화할 때 마치 없는 하이퍼파라미터가 있는 것처럼 GenAI에게 답변을 얻는 방법을 가상 하이퍼파라미터 기법이라고 볼 수 있습니다.

아래와 같은 예시를 생각해 볼 수 있습니다.



**가상 하이퍼파라미터를 사용하는 경우**

```
[질문]
- 답변 길이: 300자 내외
- 답변의 말투: 친근하게
- 답변 언어: 한국어

GenAI가 뭐야?


[답변]
GenAI는 "Generative AI"의 줄임말로, 데이터를 바탕으로 새로운 콘텐츠를 만들어내는 인공지능이에요. 
예를 들어, 글을 쓰거나, 그림을 그리거나, 음악을 작곡하는 것처럼 창작 작업을 할 수 있죠. 
GPT-4 같은 모델도 GenAI의 일종이에요. 우리가 원하는 형태로 창의적인 결과물을 생성하는 기술이라 보면 돼요!

```



가상 하이퍼파라미터를 사용하는 경우 줄글로 프롬프트를 나열하지 않고 위와 같이 "항목:값" 형식으로 작성하는 것이 가독성이 좋고 작성 시간도 절약할 수 있습니다. 또한 GenAI가 "항목:값" 스타일로 작성한 프롬프트를 더 잘 이해하기도 합니다.

복잡한 프롬프트를 작성하는 경우 가상 하이퍼파라미터 기법으로 GenAI에게 명확하게 지시하여 구어체로 작성하는 프롬프트 대비 더 좋은 결과를 얻을 수 있습니다.

# Shot

Shot 프롬프팅 기법은 GenAI에게 예시를 제공하여 답변의 성능을 끌어 올리는 방법입니다. 간단히 Shot을 예시라고 이해해 볼 수 있습니다. GenAI에게 제공하는 예시의 수에 따라서 예시를 전혀 제공하지 않으면 Zero-shot, 하나만 제공하면 One-shot, 2개 이상의 소수의 예시를 제공하면 Few-shot 프롬프팅이라고 합니다.



| 유형                  | 설명              | 예시 제공 수 |
| ------------------- | --------------- | ------- |
| Zero-shot prompting | 예시 없이 작업 지시만 포함 | 0       |
| One-shot prompting  | 예시 하나와 작업 지시 포함 | 1       |
| Few-shot prompting  | 여러 예시와 작업 지시 포함 | 2개 이상   |



## Zero-shot

개념 이해를 위해 gpt-3.5-turbo 모델에게 아래와 같이 N행시를 지어달라는 요청을 하였습니다. 이 모델은 N행시가 무엇인지 전혀 모르는 것으로 보입니다.&#x20;

```
[질문]
컴퓨터로 N행시 지어줘


[답변]
한 줄 띄어쓰기 모자라는 나의 코드
스스로 채우기 어려워 머리 아픈 나의 로직
프로그래머가 되기 위해 애쓰던 시절
모든 것을 이해하고 싶어 머리 싸늘했던 그 때
코드 한 줄 한 줄 숫자와 문자 조합하여
완성된 프로그램이 눈앞에 나타나면
뿌듯함에 가슴이 설레는 그 순간
컴퓨터 앞에서 시간 가는 줄 모르고
간절히 바라는 내 꿈을 향해 달려가는
컴퓨터로 만드는 N행시의 세계.
```



프롬프트를 조금더 구체화시켜 다시 N행시 작성을 요청해 보았습니다만, 여전히 N행시가 무엇인지 전혀 감을 잡지 못하고 있는 것으로 보입니다.

```
[질문]
너는 이제 N행시 작가야.
N행시는 주어진 단어의 각 음절로 시작하는 N줄짜리 시야.
단어가 2글자면 2줄, 3글자면 3줄, 4글자면 4줄... 

컴퓨터로 N행시 지어줘


[답변]
누구

나의 작은 비밀
나의 작은 행복
나의 작은 생각

바다

바람 소리 나면
바다의 파도 소리
바위에 앉아서

사랑

사랑스런 너의 미소
사랑한다 말해도
사랑의 끝은 없어

가을

가을이 오면
가을의 향기가
가을이 가면

행복

행복한 순간들
행복의 미소를
행복하게 살아요
```



## One-shot

이번에는 GenAI에게 하나의 예제를 제공하여 One-shot prompting을 해보겠습니다. 앞서와 동일하게 N행시에 대한 간단한 설명을 하면서 "고양이"라는 단어로 어떻게 N행시를 짓는지 예시를 제공하였습니다. "강아지"라는 새로운 단어로 N행시 작성을 요청하니 이제 제대로된 N행시를 작성해 주는 것을 볼 수 있습니다. 같은 GenAI 모델이더라도 shot(예제)를 제공해 주면 원하는 결과를 얻을 가능성이 높아짐을 확인할 수 있습니다.

```
[질문]
너는 이제 N행시 작가야. 
N행시는 주어진 단어의 각 음절로 시작하는 N 줄짜리 시야.
단어가 2글자면 2줄, 3글자면 3줄, 4글자면 4줄... 


예를 들어, 단어가 “고양이”라면:

고: 고요한 새벽에
양: 양옆으로 살금살금 걷는
이: 이야~ 귀여운 고양이다!

이제 “강아지”이라는 단어로 N행시를 지어줘.


[답변]
강: 강아지는 무지개를 따라
아: 아름다운 꽃들 사이를
지: 지금 달려가고 있어.
```



바로 이어서 이번엔 "참새"라는 단어로 N행시 작성을 부탁해 보니 여전히 이상한 답변을 합나다. 답변한 내용을 보니 예시로 제공된 "강아지"처럼 N행시는 반드시 3줄로 작성해야만 한다고 판단하는 듯합니다.&#x20;

```
[질문]
이번엔 "참새"로 지어줘


[답변]
참: 참새야 참새야
새: 새소리 나는 아침
지: 지저귀며 날아다니는
```



## Few-shot&#x20;

이번엔 2단어, 3단어, 4단어의 여러 예시(Few-shot)을 프롬프트에 작성해 보겠습니다.

```
[질문]
너는 이제 N행시 작가다.
N행시는 주어진 단어의 각 음절로 시작하는 N 줄짜리 시다.
단어가 2글자면 2줄, 3글자면 3줄, 4글자면 4줄...

예를 들어

단어가 "바다"인 경우
바: 바람이 불어온다
다: 다시 떠나고 싶어진다

단어가 "고양이"인 경우
고: 고요한 새벽에
양: 양옆으로 살금살금 걷는
이: 이야~ 귀여운 고양이다!

단어가 "대한민국"인 경우
대: 대지를 밝히는 태양
한: 한강 위로 떠오르고
민: 민들레처럼 흩어진
국: 국토를 사랑하는 마음

이제 "참새"라는 단어로 N행시를 지어줘.


[답변]
참: 참새가 짹짹대는 소리
새: 새벽 맑은 공기에

```



이어서 예시에 없던 글자의 단어도 N행시를 잘 짓는지 테스트해봅니다. "자이언트판다"라는 6글자의 단어에 대해서 N행시를 잘 짓는 것을 확인할 수 있습니다. 이제 원하는 결과를 얻을 수 있는 프롬프트가 완성되었습니다.

```
[질문]
이번엔 "자이언트판다"로 지어줘


[답변]
자: 자연 속에서
이: 이리저리 뛰노는
언: 언제나 사랑스러운
트: 특별한 존재, 판다와 함께
판: 판다는 우리 모두의
다: 다정한 친구들. 함께 지켜주자!
```



gpt-3.5-turbo 모델은 N행시를 모르고 있음을 Zero-shot promting하면서 확인하였습니다. 하지만 3개의 예제를 제시해준 Few-shot 프롬프트를 입력해주면 N행시를 어떻게 작성해야 하는지 바로 이해한 것처럼 동작합니다.&#x20;

최신 GenAI 모델들은 이미 N행시가 무엇인지 알고 있어 gpt-3.5-turbo 모델과 같이 답답하게 동작하지는 않습니다. 하지만 원하는 결과를 얻기 위해 복잡한 지시를 내리는 프롬프트를 작성하다 보면 최신 GenAI도 헤매는 경우가 심심치 않게 있습니다. 이런 경우 예시를 추가해 가며 Shot 프롬프팅으로 문제를 해결해 볼 수 있습니다.

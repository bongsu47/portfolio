# 미소천사가 알려주는 IT 기초 지식

요즘은 누구나 앱을 만들 수 있는 시대입니다.

하지만 IT에 대한 기초 지식이 없으면, 사소한 지점에서도 흐름이 끊기고 예상치 못한 어려움에 자주 부딪히게 됩니다.

이번 시리즈에서는 기초 지식이 없는 비개발자도 부담 없이 이해할 수 있도록, 실제로 업무에 바로 도움이 되는 IT 기초만을 골라 담았습니다.

이 글이 여러분의 시작의 부담을 덜어주는 안내서가 되기를 바랍니다.

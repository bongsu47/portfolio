2.  코드 노드를 아래와 같이 구성합니다.

    1. 설명 : 변수 추출기에서 각각 출력된 변수를 하나의 문자열로 묶어서 반환합니다.
    2. 입력 변수
       1. satisfaction : 변수 추출 노드의 해당 출력 변수 할당
       2. summary : 변수 추출 노드의 해당 출력 변수 할당
    3. 코드

    <pre data-expandable="true"><code>import json

    def main(satisfaction, summary):
        return {
            "result": json.dumps(
                [satisfaction, summary],
                ensure_ascii=False
            )
        }
    </code></pre>

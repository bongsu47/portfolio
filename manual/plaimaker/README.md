# 플레이메이커

## 플레이메이커 소개

플레이메이커(PLAI MAKER)는 사용자의 아이데이션을 구체화하고, 아이디어를 실제 MISO 앱으로 완성하는 AI 에이전트 오케스트레이션입니다.

### 각 분야의 전문 에이전트와 함께하세요

아이디어가 아직 막연해도 괜찮습니다. 플레이메이커의 에이전트들이 함께 아이디어를 구체화하고, 기획부터 앱 제작까지 전 과정을 함께 완성합니다.

### 무엇을 만들 수 있나요?

플레이메이커는 MISO 앱 중 **워크플로우, 에이전트 타입** 생성을 지원합니다.\
**일반 워크플로우**와 **대화형 워크플로우, 에이전트** 모두 만들 수 있으며, 생성된 앱 기존 MISO 앱과 동일하게 **앱 리스트에 등록**됩니다.\
생성 이후에도 필요에 따라 **자유롭게 편집**할 수 있습니다.

다만 MISO에서 **기본으로 지원하지 않는 기능은 구현에 제한**이 있을 수 있습니다.

{% hint style="warning" %}
* 플레이메이커는 **관리자가 활성화한 경우에만** 사용할 수 있습니다.\
  만약 MISO에서 플레이메이커가 보이지 않는다면, **관리자에게 활성화 여부를 문의해 주세요.**
{% endhint %}

# 웹 검색

웹 검색 도구 기반 검색 엔진을 사용하여 실시간 웹 검색을 통해 질문에 대한 답변, 요약 정보, 참조 URL을 제공합니다.&#x20;

최신 정보 조회, 일반적인 웹 검색, 여러 출처의 정보 종합이 필요할 때 사용할 수 있습니다.\
사용된 정보의 출처는 메시지 하단에 '레퍼런스 리스트' 형태로 제공되어 원본 정보의 출처를 확인할 수 있습니다.



* **메시지 인풋창 하단의 ‘+’ 버튼을 통해 웹검색 도구 활성화**\
  (단, 비활성화시에도 미소 AI가 웹검색이 필요한 요청으로 판단할 경우 자동 활성화될 수 있음)
* **응답 메시지 내용 및 하단 웹검색 출처 정보(링크) 확인 가능**

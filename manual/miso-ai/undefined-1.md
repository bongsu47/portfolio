# 이미지 생성

이미지 생성 도구를 활용하여 텍스트 프롬프트를 기반으로 새로운 이미지를 생성하거나, 업로드된 이미지의 색상·배경·스타일 등을 편집할 수 있습니다.&#x20;

이미지 생성을 요청하거나 기존 이미지의 수정이 필요할 때 사용하며, 수정 시에는 원본 이미지를 전달하고 변경 사항을 프롬프트로 구체적으로 설명해야 합니다.&#x20;

스타일, 색상, 구도, 분위기 등을 상세히 기술할수록 더 정확한 결과를 얻을 수 있습니다.



* **메시지 인풋창 하단의 ‘+’ 버튼을 통해 이미지 생성 도구를 활성화**
* **이미지 생성 요청 프롬프트 작성 및 메시지 전송**
* **생성에 참고 or 수정할 기존 이미지가 있을 경우 "사진 및 파일 추가"를 활용하여 업로드**

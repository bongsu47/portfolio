# 워크플로우란?

워크플로우는 **여러 기능을 순서대로 연결해 원하는 작업을 자동으로 처리하는 앱**입니다. **블록을 조립하듯 노드(Node)를 이어 작업 흐름을 만들 수 있고**, 각 노드는 **AI 호출, 지식 검색, 조건 분기처럼 한 가지 기능을 담당**합니다. 그래서 **복잡한 작업도 정해진 순서에 따라 쉽게 실행할 수 있습니다.**

워크플로우는 채팅 설정 여부에 따라 두 가지 방식으로 사용할 수 있습니다.

{% include "../../.gitbook/includes/workflow-create-mode.md" %}

아래 기능은 채팅 설정을 켜야 사용할 수 있습니다.

| 옵션           | 설명                    |
| ------------ | --------------------- |
| **시작 시 채팅창** | 대화형 인터페이스             |
| **환영 메시지**   | 대화 시작 시 보여줄 인사말       |
| **질문 제안**    | AI가 다음에 할 수 있는 질문을 추천 |
| **메모리**      | LLM 노드에서 이전의 입력내용을 기억 |

# 보안

<figure><img src="../../../.gitbook/assets/image (444).png" alt=""><figcaption><p>보안 설정</p></figcaption></figure>

**가드 레일**: 앱 입력 또는 출력 내용의 안정성, 적절성, 규정  준수를 보장하기 위한 가드레일 및 키워드 필터링시스템입니다.(Default는 관리자에 의해 관리되며, 관리자가 선택적으로 가드레일 설정 강제를 해지하면, 사용자가 컨트롤 할 수 있습니다.)

**차단 모드**: 가드레일 행위에 감지되었을 때, 메시지 차단을 할 것인지에 대한 설정 입니다.

**키워드 필터**: 특정 키워드를 감지하여, 필터링 하는 기능 입니다.

**차단 시, 대체 메시지**: 차단 모드 활성화로 대화 중 가드레일 정책에 의해 차단이 되었을 때, 차단 메시지로 대체 할 것인지에 대한 선택 옵션 입니다.

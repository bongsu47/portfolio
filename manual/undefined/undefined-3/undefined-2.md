# 파일

<figure><img src="../../../.gitbook/assets/image (442).png" alt=""><figcaption><p>파일 설정</p></figcaption></figure>

**파일 입력**: 사용자 질문에 파일을 업로드 하여, 사용할 수 있도록 제공되는 기능입니다.

<figure><img src="../../../.gitbook/assets/image (443).png" alt=""><figcaption><p>파일 업로드 설정</p></figcaption></figure>

**입력 방식**: 로컬 파일에서 업로드만 적용할 지, URL을 통한 입력을 할 지, 또는 둘다 가능하게 할 것인지 선택하는 옵션 입니다.

**개수 제한**: 최대  업로드 가능한 파일의 개수를 제한합니다.&#x20;

**파일 유형**: 문서 또는 이미지 파일을 제공 합니다.

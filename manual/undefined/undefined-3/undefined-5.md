# 자동 실행

원하는 시점과 주기에 맞춰 MISO 앱이 지정한 작업을 자동으로 실행하는 결과를 알려주는 기능 입니다.

<figure><img src="../../../.gitbook/assets/image (431).png" alt=""><figcaption><p>자동 실행 On</p></figcaption></figure>

<figure><img src="../../../.gitbook/assets/image (430).png" alt=""><figcaption><p>자동 실행 - 설정</p></figcaption></figure>



* **일정**: 반복하지 않음/매년/매월/매주/매일 중 택 1로 실행 시간에 맞춘 설정 값 대로 아래 동작을 수행합니다.
* **실행 시간**: 주기 설정 후, 동작 되는 시간을 정의 합니다.
* **대화**: 앱에 입력할 채팅 내용을 정의합니다.

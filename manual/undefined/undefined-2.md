# 도구 설정

<figure><img src="../../.gitbook/assets/image (436).png" alt=""><figcaption><p>도구 설정</p></figcaption></figure>

**편집**: 에이전트에서 사용할 도구를 선택하기 위한 팝업이 호출 됩니다.



<figure><img src="../../.gitbook/assets/image (437).png" alt=""><figcaption><p>도구 설정</p></figcaption></figure>

**도구**: MISO에서 제공하는 기본 도구입니다.

**커스텀 도구**: MISO에 사용자 등록한 종류의 도구 입니다.

**워크플로우**: MISO 워크플로우 형식으로 등록된 도구의 종류 입니다.

**MCP**: 해당 워크스페이스에 등록된 MCP 도구 목록 입니다. ( MCP 등록 방법 참조 [링크](../tools/mcp.md) )

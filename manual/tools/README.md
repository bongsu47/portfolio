# 도구

<figure><img src="../../.gitbook/assets/image (191).png" alt=""><figcaption><p>&#x3C;도구 둘러보기></p></figcaption></figure>

도구는 MISO에서 외부의 시스템과 연동이 필요할 경우 사용하는 기능 입니다. 제공하는 형식과 주체에 따라 기본 도구 / 사용자 정의 / 모듈로 구분 합니다



**기본 도구**

* MISO에서 연결을 제공하는 도구로 자주 사용되는 도구에 한하여 MISO가 미리 준비해 놓은 도구 입니다.

**커스텀 도구**

* 사용자가 외부의 시스템을 JSON 또는 YAML 형태에 맞춰 등록하여 사용하는 도구 입니다.
* 스키마 정의에 의해 등록된 도구 리스트를 호출 하고, 호출 된 도구를 이용할 수 있습니다.

**워크플로우**

* MISO의 워크플로우를 구현한뒤 이를 도구로 만들 수 있습니다.
* 워크플로우로로 생성된 도구도 다른 도구와 동일하게 에이전트와 워크플로우에서 도구로 사용될 수 있습니다.

**MCP**

* MCP 서버 정보를 입력하여 해당 MCP 서버를 도구로 등록할 수 있습니다.

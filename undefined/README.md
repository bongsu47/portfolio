{% hint style="info" icon="bullhorn" %}
**\[도입 가이드 목표]**

처음 MISO를 도입하는 <kbd>???</kbd>에게, 다음의 온보딩을 제공하고자 합니다.

1.  **앱 활용**

    기본앱 세팅을 통해 MISO 앱 예시를 살펴보며, _어떤 작업을 MISO 앱으로 구현할 수 있는지, 구현은 어떻게 할 수 있는지_ 전달드리고자 합니다.
2.  **쇼케이스 활용**

    조직 내 구성원들이 공통적으로 활용할 수 있는 `인기 템플릿` 설정 방법과, 구성원들이 `가장 많이 쓰는 앱`을 표시하는 방법을 전달드리고자 합니다.
{% endhint %}

{% hint style="info" icon="arrow-progress" %}
**\[진행 흐름]**

1. 첨부해드린 기본 앱과, 첨부파일을 **다운**받습니다.
2. 기본앱 상세 설명 페이지를 순서대로 읽어보며 MISO에 앱과 첨부 파일을 **등록**하고, **테스트** 해봅니다.
3. 쇼케이스 등록을 위해 앱 **공유 범위**를 설정합니다.
4. Admin 계정을 활용해 구성원이 모두 확인할 수 있는 **쇼케이스** 페이지에 앱을 등록해봅니다.
{% endhint %}

진행 과정 중에 모르겠는 부분이 생기시면 Slack @MISO Support 챗봇 혹은 **MISO팀**([miso@gs.co.kr](mailto:miso@gs.co.kr))에 문의해주세요!

## 1. 기본앱 세팅

기본앱 세팅은 다음의 2가지 목적을 가지고 있습니다.

1. **전사 관점에서 미소에 쉽게 온보딩**
   * 기본 앱을 통해 “어떤 앱들을 만들어서 어떤 문제들을 해결/자동화 할 수 있는지” 예시를 보여드리고자 합니다.
2. **MISO 앱 활용에 대한 튜토리얼**
   * 앱을 만들 수 있는 2가지 방법, 1) 워크플로우, 2) 에이전트 제작 예시를 설명합니다.
   * MISO에서 제공하는 `도구`를 활용하는 방법과, 자체 보유 데이터를 MISO에 `지식`으로 등록해 활용하는 예시를 다루며, MISO를 활용하는 다양한 방식을 안내하고자 합니다.



{% hint style="info" icon="lightbulb" %}
미소 내의 “학습하기” 매뉴얼 만으로는 MISO의 활용 흐름을 체감하기 어려운 부분이 있어, 실제 동작하는 앱을 통해 활용 시나리오를 직접 경험해보실 수 있도록 기본앱을 세팅합니다.
{% endhint %}

## 2. 쇼케이스 앱 등록

\<aside> ⚠️

**여기서부터는 Admin 계정**에서 진행해주세요!

\</aside>

`쇼케이스`는 **조직 내에서 활용 가능한 앱**들을 한눈에 살펴볼 수 있도록 구성된 메인 랜딩 페이지입니다.

조직 차원에서 1) 권장하는 표준 앱과, 2) 구성원이 직접 만들어 활발히 사용되고 있는 앱을 함께 보여줌으로써, 사용자가 자신의 업무에 맞는 앱을 쉽게 발견하고 활용할 수 있도록 돕습니다.

쇼케이스는 다음 2개의 섹션으로 구성됩니다.

1.  **인기 템플릿**

    일반 업무에 두루 적용할 수 있는 `공용 기본앱`이 노출되는 섹션으로, 조직 차원에서 권장하는 대표 앱들로 구성됩니다.

    \<aside> ℹ️

    **Admin 계정에서 업로드한 앱만** 이 섹션에 등록됩니다.

    \</aside>
2.  **동료가 만든 앱**

    구성원이 직접 제작해 전체 공유한 앱이 표시되는 섹션으로, 현업에서 만들어진 실무 앱들이 노출됩니다.

    \<aside> ℹ️

    **사용량 기준 내림차순으로 정렬**되어, 조직 내에서 가장 활발히 활용되고 있는 앱을 한눈에 확인하실 수 있습니다.

    \</aside>

#### 1단계: 쇼케이스 앱 등록

![image.png](attachment:73d4dd3b-4775-4826-adce-83ef32af1029:image.png)

`쇼케이스` 기본 페이지 중 `인기 템플릿` 탭에 **법령 검색 에이전트**를 추가해보겠습니다.

![image.png](attachment:086a83a9-ba45-44c7-894c-8b48ca834757:image.png)

좌측 패널에서 쇼케이스 를 클릭하여 들어가줍니다.

![image.png](attachment:daf3daa5-31d8-4572-a5ba-e80d8dda28e8:image.png)

스크롤을 내려 **법령 검색 에이전트**를 찾은 뒤, 오른쪽 `노출 상태` 체크박스를 체크해줍니다.

![image.png](attachment:b924dbd0-a3dd-44ce-bc17-b9153348ee41:image.png)

1. `변경사항 저장` 을 해준 뒤
2. `쇼케이스 작업 실행` 을 눌러줍니다.

\<aside> 💡

변경사항 저장 : **관리자 계정**에서 확인할 쇼케이스 목록 설정

쇼케이스 작업 실행 : **모든 계정**에서도 쇼케이스 화면에서 볼 목록을 Update 해주는 버튼

* **목록 뿐만 아니라, ‘사용량’ 에 따른 노출 순위를 Update 할 때도 사용하기 때문에 2개의 버튼은 구분되어 있습니다.** \</aside>

![image.png](attachment:c73fab75-b4c9-49f7-99b4-795a626a294b:image.png)

이후 각 계정별로 `쇼케이스` 에 들어가보면 **법령 검색 에이전트**가 인기 템플릿 영역에 추가된 것을 확인할 수 있습니다.

#### 2단계: 쇼케이스 업데이트

**option 1 : 자동 업데이트**

![image.png](attachment:ff22559d-3dcd-46df-a9ec-b03b669a8fca:image.png)

쇼케이스 작업 실행 버튼을 누르면, 사용량을 기준으로 ‘동료가 만든 앱’ 상위 11개가 자동 선택됩니다.

이때, 인기 템플릿은 Admin이 직접 선택한 것이기 때문에 자동 선택 범위에서 제외되며, 기존에 설정한 상태 그대로 유지됩니다.

**option2 : 수동 업데이트**

![image.png](attachment:082190bd-45ea-4488-9a58-f0011ca46841:image.png)

우측의 `노출 상태` 체크 박스를 통해서 쇼케이스 화면에서 보여주고 싶은 앱 목록을 수동으로 선택할 수 있습니다.

![image.png](attachment:193ec789-1adf-4648-9725-8cc0df3f800a:image.png)

이후, 좌상단의 `변경사항 저장` 을 클릭하면, 쇼케이스 화면에 반영됩니다.

\<aside> ⛳

**\[쇼케이스 Checklist]**

* \[ ] 앱을 완성하고 `발행`까지 완료했나요?
* \[ ] 구성원 모두가 볼 수 있도록 `공유 범위 설정`을 **모든 워크스페이스**로 설정했나요?
* \[ ] `관리자 승인`을 받았나요? \</aside>
